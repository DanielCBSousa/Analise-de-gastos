# 📊 Análise de Gastos Pessoais

Projeto desenvolvido em Python para análise e visualização de gastos pessoais a partir de um arquivo CSV.

---

## 🚀 Funcionalidades

* Leitura de dados de um arquivo CSV
* Cálculo do total de gastos
* Cálculo da média de gastos
* Identificação do maior gasto
* Análise de gastos por categoria
* Geração de gráficos:

  * 📊 Gastos por categoria
  * 📈 Evolução dos gastos ao longo do tempo

---

## 🛠️ Tecnologias utilizadas

* Python
* Pandas
* Matplotlib

---

## 📂 Estrutura do projeto

```
analise_gastos
│
├── main.py
├── data
│   └── gastos.csv
├── graphs
```

---

## 📊 Exemplo de dados

```
data,categoria,valor
2026-01-01,Alimentacao,35
2026-01-02,Transporte,10
2026-01-02,Lazer,50
```

---

## ▶️ Como executar

1. Clone o repositório:

```
git clone https://github.com/DanielCBSousa/analise_gastos.git
```

2. Acesse a pasta:

```
cd analise_gastos
```

3. Instale as dependências:

```
pip install pandas matplotlib
```

4. Execute o projeto:

```
python main.py
```
